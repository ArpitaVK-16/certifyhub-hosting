Tip : Have you observed one thing, that each of the verbes given above ends with 'en'?

Yes! Almost 99.99% verbs in German end with 'en'!


Now before constructing some sentences, you have to follow a rule table given below :


Pronoun Rule Example
ich Add e to the root word trinke
du Add st to the root word trinkst
er / sie / es Add t to the root word trinkt
ihr Add t to the root word trinke
wir Add en to the root word trinken
sie / Sie Add en to the root word trinken


Now one more thing to know! German is an Europian Language like an English. In Europian Languages, while constructing a sentence, verb is at second position.


English sentence Translation in German

You play cricket Du spielst Kricket
He plays cricket Er spielt Kricket
